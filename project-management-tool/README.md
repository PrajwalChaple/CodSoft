# Project Management Tool

A full-stack project management application with a Vite frontend and an Express/MongoDB backend.

## Structure

- `client/` - frontend app
- `server/` - backend API

## Run locally

Frontend:

```bash
cd client
npm install
npm run dev
```

Backend:

```bash
cd server
npm install
npm run dev
```

Create a `server/.env` file before starting the backend.
